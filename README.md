# Marvel Mission Control

A lightweight mission board for planning, prioritizing, and celebrating daily wins. Add missions, mark them complete, and track key stats with persistent local storage.

## Quick start

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000> in your browser.

## Features

- Add missions with priority and optional due dates.
- Filter by all, active, or completed missions.
- Track active, completed, and high-priority counts at a glance.
- Persists missions in your browser's local storage.
